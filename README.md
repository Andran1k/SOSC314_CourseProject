# YouTube Trending Language Project

## Research Question
How does the language used in YouTube trending video metadata differ across content categories?

## Data Source
YouTube Data API v3 (videos.list, chart=mostPopular)

## Unit of Analysis
One YouTube trending video.

## Repo Structure
- data/raw: raw pulled data (CSV sample)
- scripts: API data collection scripts
- notebooks: exploration and analysis
- figures: figures used in reports
- reports: weekly notes/drafts
